🛠️ TROUBLESHOOTING

Issue: Colleges not loading?
  → Check Supabase credentials in public/js/supabase-client.js
  → Verify database tables exist (SQL Editor → Run schema scripts)
  → Check browser console (F12 → Console tab)

Issue: Sign up fails?
  → Ensure password is 8+ characters
  → Check email format is valid
  → Verify Supabase Auth is enabled

Issue: 404 errors on navigation?
  → Make sure server is running from project root
  → Check file paths use relative paths (/pages/...)
